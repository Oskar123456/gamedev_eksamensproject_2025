## Instructions

### Controls

  - R : [R]estart
  - WASD : Movement
  - Shift : Sprint
  - Space : Jump
  - Mouse Buttons: Attack
  - Right Mouse Button: Cast Spell
  - Number Keys : Switch Active Attack
  - [Left-Shift+]Number Keys : Switch Active Spell
  - Tab : [Maximize] Minimap
  - T : Open Skill Tree
  - I/E : Open Inventory
  - Escape : Close UI
